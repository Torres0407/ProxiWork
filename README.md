# ProxiWork

**ProxiWork** is a hybrid freelance and local gig marketplace designed for Nigeria.  
It connects digital freelancers and local service providers with clients looking for tasks, jobs, and projects.  

This project demonstrates **full-stack web development skills**, including frontend UI, backend logic, and database integration.

### For Service Providers / Freelancers

- Browse available jobs and local gigs

- Receive ratings and reviews for completed work
- Track earnings and completed tasks

### For Clients / Employers
- Post digital jobs or local tasks
- Review proposals and hire providers
- Track ongoing jobs and task status
- Chat with providers
- Rate completed tasks
- Dashboard for managing posted opportunities

### Additional Features
- Role-based login (Client / Provider)
- Secure payment simulation using Paystack integration (demo)
- Notifications for job updates and new proposals
- Categorized search (Digital Services vs Local Tasks)
- Responsive UI using Tailwind CSS
